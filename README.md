# Text Annotation API

REST API для загрузки текстов, аннотирования фрагментов и управления метками.  
Проект позволяет пользователям загружать `.txt` файлы, размечать части текста и сохранять аннотации с метками.

Реализована аутентификация по JWT-токену, постраничная отдача содержимого документа и фильтрация аннотаций по документу.

---

## Стек технологий

- Python 3.12+
- Django 5.1
- Django REST Framework
- Djoser + JWT
- SQLite (по умолчанию)
- Redoc (статическая документация)

---

## Документация API

После запуска проекта документация доступна по адресу:

```
http://127.0.0.1:8000/redoc/
```

## Как запустить проект

Клонируйте репозиторий и перейдите в директорию проекта:

```
git clone <link>
```

```
cd <имя_проекта>
```

Создайте и активируйте виртуальное окружение:

```
python3 -m venv env
```

```
source env/bin/activate
```

```
python3 -m pip install --upgrade pip
```

Установить зависимости из файла requirements.txt:

```
pip install -r requirements.txt
```

Выполнить миграции:

```
python3 manage.py migrate
```

Запустить проект:

```
python3 manage.py runserver
```
