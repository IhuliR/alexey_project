# Статическое демо (GitHub Pages)

Эта папка содержит полностью статическое многостраничное демо на чистых HTML, CSS и Vanilla JS.
Оно предназначено для публикации через GitHub Pages из ветки `master` и папки `/demo`.

## Публикация на GitHub Pages

1. Отправьте репозиторий в GitHub.
2. Откройте настройки репозитория -> **Pages**.
3. В разделе **Build and deployment** выберите:
   - **Source**: Deploy from a branch
   - **Branch**: `master`
   - **Folder**: `/demo`
4. Сохраните настройки и дождитесь публикации GitHub Pages.

Формат опубликованного URL:

```text
https://<username>.github.io/<repo>/
```
